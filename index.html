Entropy, not just complexity

Password strength is about unpredictability. Entropy is measured in bits: \(E = L \times \log_2(R)\), where \(L\) is length and \(R\) is the size of the character set. Long passwords or multi‑word passphrases usually beat short “complex” ones.

Checking breaches privately

The breach checker uses Have I Been Pwned’s range API. Your password is hashed locally, and only the first 5 characters of the hash are sent. The final comparison happens in your browser.

Random vs Diceware

Random passwords pack more entropy per character but are hard to remember. Diceware passphrases trade some density for memorability by combining random words that your brain can recall more easily.

Accessibility as a security feature

Clear feedback, readable text, keyboard support, and strong contrast make it easier for everyone to use secure passwords — including people using assistive technologies.
